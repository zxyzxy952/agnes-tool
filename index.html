获取 API Key 教程

Agnes AI 提供免费的多模态 API，支持文字对话、文生图、图生图、文生视频和图生视频。只需注册账号并获取 API Key，即可在本工具中使用全部功能。

1

打开 Agnes AI 官网并注册账号

点击下方按钮打开 Agnes AI 开发者平台：

打开 platform.agnes-ai.com

在页面上点击 Sign Up（注册）按钮，使用邮箱注册一个新账号。注册完成后登录。

  • 支持 Google 账号快捷登录
  • 也可以用任意邮箱注册
2

进入 API Keys 管理页面

登录后，在左侧菜单栏找到 API Keys 选项并点击进入。你会看到一个管理 API 密钥的界面。

  • 如果是新注册的账号，页面上会是空的，这是正常的
3

创建新的 API Key

点击页面上的 Create New Key（创建新密钥）按钮。系统会弹出一个窗口，你可以：

  • 给密钥起一个名字（比如"手机工具"），方便你以后辨认
  • 直接点确认创建

创建成功后，会显示一串以 sk- 开头的字符串，这就是你的 API Key。

sk-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
4

复制并粘贴到工具中

点击 API Key 旁边的复制按钮，把密钥复制到剪贴板。然后回到本工具页面，在顶部的输入框中粘贴你的 API Key：

粘贴到「粘贴你的 Agnes API Key」输入框中

粘贴完成后，所有功能（对话、生图、视频）就可以直接使用了。

5

开始使用

API Key 只需填写一次，之后每次打开工具会自动保存（如果你的浏览器支持）。现在你可以：

  • 💬 对话 — 跟 AI 聊天、问问题
  • 🖼️ 文生图 — 输入文字描述，生成图片
  • 🎨 图生图 — 上传图片 + 描述，AI 帮你改造
  • 🎬 文生视频 — 输入描述，生成约 5 秒视频
  • 📹 图生视频 — 上传一张照片，让 AI 让它动起来

⚠️ 安全提醒

  • API Key 是你的个人密钥，不要分享给任何人
  • 不要把 API Key 发到公开群聊、论坛或社交媒体上
  • 如果不小心泄露了，去平台删除旧的 Key 并重新创建一个
  • Agnes 目前提供免费额度，但请合理使用，不要滥用
